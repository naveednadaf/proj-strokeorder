# Chinese Character Stroke Animator

A web application that animates Chinese character stroke order with integrated dictionary lookup.

## Features

- **Stroke Order Animation**: Watch Chinese characters being written stroke by stroke
- **Loop Animation**: Characters animate continuously for easy learning
- **Adjustable Speed**: Control animation speed from 0.5x to 3x
- **CC-CEDICT Dictionary**: Integrated Chinese-English dictionary with:
  - Pinyin with proper tone marks (e.g., nǐ, hǎo)
  - English definitions
  - Toggleable dictionary panel

## Usage

1. Open `index.html` in a web browser
2. Enter Chinese characters in the input field (e.g., 你好世界)
3. Click **Generate** or press Enter
4. Click **Eng + Pinyin** on any character card to view dictionary information
5. Use the speed slider to adjust animation speed
6. Click **Reset** to restart the animation

## Technologies

- **HanziWriter**: Stroke order animation library
- **CC-CEDICT**: Chinese-English dictionary data
